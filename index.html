🤖 AI Meeting Notes Summarizer

A full-stack application that transforms meeting transcripts into structured summaries using AI.

🚀 Launch Demo

✨ Features

📝 Smart Summarization: Upload transcripts and get AI-powered summaries
🎯 Custom Prompts: "Summarize for executives" or "Extract action items"
✏️ Editable Results: Modify summaries before sharing
📧 Email Sharing: Send summaries to multiple recipients

🛠️ Tech Stack

  • Frontend: React 19 + Vite
  • Backend: Node.js + Express
  • AI: Groq API (Llama3)
  • Email: Nodemailer
